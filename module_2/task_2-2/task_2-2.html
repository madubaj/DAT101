1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Constants should be used only where variables (let) is not possible to use, and where the value cannot be changed. Not all values can be constant- some values need to change with the data.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses change the order of the operation, meaning parentheses are performed/evaluated before other parts of an expression.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. By using the to.Fixed() method, and writing the given number of decimals inside (), e.g. toFixed(2) for two decimals.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. To see if two floating-point values are equal, we can compare them with a tolerance value (Epsilon). We use the function Math.abs() and set the epsilon value to e.g. 0.000001:

    const EPSILON = 0.000001;
    printOut(Math.abs(12.199999 == 12.2) < EPSILON);

    This calculates the difference between the two numbers, and then compares the difference to the epsilon. The numbers are equal if the difference is smaller than the epsilon.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return boolean values. To check if they're false or true, we can use if statements.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() gives a random number between 0 and 1, always lower than 1 and never exactly 1. It can be used for generating random numbers between 0 and 1, generating a random number between two values etc.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus is an operator which returns the division remainder - what's "left over" of a perfect division. This is useful when we want to determine the remainder of a division operation.
